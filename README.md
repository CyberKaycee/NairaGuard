# NairaGuard — Real-Time Price Intelligence & Margin Protection

NairaGuard is a high-utility mobile platform designed to protect profit margins for merchants, traders, and vendors across major Lagos markets. By compiling localized commodity indices and offering cross-market pricing intelligence (Mainland vs. Island valuations), the application equips retail businesses with the data needed to hedge against sudden currency and market fluctuations.

---

## 🛠️ Project Roadmap & Development Stages

To maintain structural discipline and focus entirely on engineering core feature mechanics, the platform deployment is systematically broken down into three primary phases:

1. **MVP Stage (Current):** Focused strictly on core value validation—real-time price processing engines, localized market data sync, and premium tier validation mechanisms to present a lean demonstration for investor pitch funding.
2. **BETA Stage (Next):** Private, controlled user onboarding launched via the **Google Play Store Closed Testing Track** to validate system stability across a restricted whitelist of vetted local merchants.
3. **Commercial Scale Stage:** Full multi-user registration, cloud token authorization, comprehensive analytics metrics, and live payment processing integration.

## 🚀 Step-by-Step Installation

Follow these instructions to download and install the standalone NairaGuard application package directly onto an Android device:

### 1. Enable Installation from Unknown Sources
Because this is a standalone pre-release package not yet published on the public app store, your phone requires authorization to run it:
* Go to your phone's **Settings > Apps > Special app access**.
* Tap **Install unknown apps**.
* Select the browser or file manager you will use to download the file (e.g., *Google Chrome* or *Mi File Manager*) and toggle **Allow from this source** to **ON**.

### 2. Download the Application Package
* Locate the compiled **`NairaGuard.apk`** file provided in this release bundle (or use the designated download link provided by the system administrator).
* Save the `.apk` file directly into your device's internal storage or **Downloads** folder.

### 3. Run the Installer
* Open your device’s built-in File Manager application and navigate to the **Downloads** directory.
* Tap on the **`NairaGuard.apk`** file.
* A system prompt will appear asking: *"Do you want to install this application?"*
* Tap **Install**. 

### 4. Complete Device Verification (For Xiaomi / Redmi Devices)
* If your device runs Xiaomi HyperOS or MIUI, an overlay security scan may appear. 
* Ensure your screen is unlocked, acknowledge any pre-release development notifications, and tap **Allow** or **Open** to launch the application.
* The NairaGuard icon will now appear permanently on your home screen grid, ready for operational testing.

## 🔒 Security & Admin Architecture

The application repository supports dual-interface deployment environments:
* **Vendor/Merchant Facing App:** Integrates data matrices tracking localized commodity price updates across key regional distribution hubs (e.g., Mile 12, Isale Eko, Ikorodu, Agege, Ketu, and Lekki).
* **Super Admin Command Center:** A secure, dedicated administrative terminal enabling system operators to manually override pricing entries, monitor real-time user registrations, track active subscription clusters, and manage tenant licensing states.
